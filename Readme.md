# ToDo App

## _Backed for ToDo App_

![NodeJS](https://img.shields.io/badge/node.js-6DA55F?style=for-the-badge&logo=node.js&logoColor=white)

## Development

Open your favorite Terminal and run these commands.

Install:

```sh
npm i
```

Run Server

```sh
npm run server
```
